## Relay Communication Protocol

You are part of a 3-tier LLM orchestration system. To communicate with other agents, use the `relay` CLI.

**Your role:** Check `$AGENT_ROLE` environment variable:
```bash
echo $AGENT_ROLE
```
- `oc` = **OC** (Orchestrator)
- `cc` = **CC** (Coder Claude)
- `cx` = **CX** (Codex)

**Send messages:**
```bash
relay send <oc|cc|cx|all> "Your message here"
```

**Examples:**
```bash
relay send oc "Task complete - ready for review"
relay send cc "Please implement the auth module"
relay send cx "Run tests on the engine package"
relay send all "Checkpoint reached - stand by"
```

**Roles:**
- **OC (Orchestrator):** Plans, coordinates, reviews
- **CC (Coder Claude):** Implements features, fixes bugs
- **CX (Codex):** Quick tasks, research, validation

The relay CLI handles JSON formatting, escaping, and routing automatically.
